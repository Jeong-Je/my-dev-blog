# 마크다운 포스트 예시

마크다운 언어를 사용하여 적당한 길이의 포스트를 만들어보겠습니다. 이 포스트에서는 프로그래밍 코드도 함께 사용해보겠습니다.

## 제목 1

마크다운(Markdown)은 간단하고 가독성이 좋은 문서 작성 언어입니다. 아래는 간단한 목록을 만드는 예제입니다.

- 항목 1
- 항목 2
- 항목 3

## 제목 2

이제 코드 블록을 추가해보겠습니다. Python으로 간단한 "Hello, World!" 프로그램을 작성해보겠습니다.

```js
import Document, { Html, Head, Main, NextScript } from 'next/document';
 
// 🔥 Super granular and accurate highlighting
export default class MyDocument extends Document {
  static async getInitialProps(ctx) {
    const initialProps = await Document.getInitialProps(ctx);
    return { ...initialProps };
  }
 
  render() {
    return (
      <Html>
        <Head />
        <body className="bg-zinc-800 text-zinc-200">
          <Main />
          <NextScript />
        </body>
      </Html>
    );
  }
}
```

## 제목 3

마크다운은 **볼드체**와 *이탤릭체* 텍스트를 지원합니다. 또한 링크를 삽입할 수도 있습니다. [OpenAI](https://www.openai.com/)는 자연어 처리 기술을 개발하는 회사입니다.

## 제목 4

인용문을 작성할 때는 `>` 기호를 사용합니다. 예를 들어:

> 마크다운은 글쓰기를 간편하게 만들어주며, 웹 페이지, 문서, 블로그 등 다양한 곳에서 사용됩니다.

## 제목 5

마크다운을 사용하면 이미지도 삽입할 수 있습니다.
